## Steps to Run the App
### 1. Clone the project

### 2. Run the project
```shell
npm i
npm start
```

### 3. Install the `Moesif CORS \& CORS Changer' extension for your web browser
  - Turn on when using app

### 4. Press the 'Get Started' button to be directed to the Keyboard UI
